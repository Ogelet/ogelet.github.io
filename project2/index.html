Rules of the Match-Match-Game

Match-match-game is easy to play, in fact it is so simple that really young children can play with ease. It requires observation, concentration and a good memory to win.

Object of the Game

The object of the game is to collect the most matching pairs.

Playing Match-match-game

  • Select two cards to try to match the pictures
  • If you match the pictures you can go again
  • If they don't match ig is the computer turn them
  • The player that finds all pairs wins
  • Have fun!
